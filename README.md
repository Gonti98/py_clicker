# PyClicker

**Terminal-only clicker game** - learning project for Python Textual TUI framework.
Everything runs in terminal with **vimlike controls**.

## Requirements
- Python ≥ 3.14.3

### Nice to have
- GPU-accelerated terminal (Alacritty, Ghostty, Kitty)
- Nerd Fonts (for ASCII art)

## Quick Start

```bash
git clone git@github.com:Gonti98/py_clicker.git && cd py_clicker

./init.sh
```
